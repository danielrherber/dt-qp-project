Welcome to DTQP's documentation!
================================

:term:`DTQP` project` solves linear-quadratic dynamic optimization (LQDO) problems using direct transcription (DT) and quadratic programming (QP).

Check out the :doc:`getting-started` section for further information, including :ref:`Installation` information.

.. note::

   This project is under active development on `GitHub <GitHub_>`_.
